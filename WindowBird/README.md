# Window Bird

A bird silhouette designed to be mounted on windows to reduce bird strikes and discourage birds from landing or leaving droppings.

![bird_on_window.jpg](images/bird_on_window.jpg)

| Setting | Value |

| Layer Height | 0.2 mm |
| Nozzle Diameter | 0.4 mm |
