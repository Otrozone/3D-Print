# Olive Necklace

Minimalist and clean design. Top hole allows easy attachment to a cord or chain.

Designed for single-extruder printers (without multimaterial upgrade). Switch filament for the top layers. Compact, durable, and easy to print, this pendant is ideal for quick prints, personalization, or as a simple gift.

[olive_neckle.jpg](images/olive_neckle.jpg)

## Print Settings

| Setting | Value |
|---|---|
| Layer Height | 0.2 mm |
| Nozzle Diameter | 0.4 mm |
| Material | PETG |

## Slicer Settings

| Setting | Value |
|---|---|
| Infill | 15% |
| Infill Pattern | Gyroid |
| Perimeters | 2 |
| Supports | No |

## Assembly Instructions

Attach a cord or chain through the top hole. A small metal ring can be used for easier attachment.
