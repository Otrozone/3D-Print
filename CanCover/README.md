# Can Cover

A simple yet nicely working sliding cover for can with diameter ~75 mm.

## Print Settings

| Setting | Value |
|---|---|
| Layer Height | 0.2 mm |
| Nozzle Diameter | 0.4 mm |
| Material | PETG |

## Slicer Settings

| Setting | Value |
|---|---|
| Infill | 15% |
| Infill Pattern | Gyroid |
| Perimeters | 2 |
| Supports | No |

## Notes

Use PETG, as it is generally considered safe for incidental food contact and offers good moisture and chemical resistance.
