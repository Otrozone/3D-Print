# Modular Moss Pole

A modular, stackable moss pole system designed to support climbing plants. Easily extendable and customizable to fit different pot sizes and plant growth stages.

## Print Settings

| Setting | Value |
|---|---|
| Layer Height | 0.2 mm |
| Nozzle Diameter | 0.4 mm |
| Material | PETG |

## Slicer Settings

| Setting | Value |
|---|---|
| Infill | 15% |
| Infill Pattern | Gyroid |
| Perimeters | 3 |
| Supports | Components 'Cap' and 'Entonnoir B' require supports |

## Notes

For indoor use, choose PETG, as it resists moisture. For outdoor use, choose ASA, as it resists both moisture and UV light.

## Assembly Instructions

Individual segments connect together to form a taller pole. Insert moss or other growing medium into the structure and secure it in a plant pot.
